# AI-Powered Loan Application Review System

## Overview

This project is an AI-powered loan application automation workflow built using n8n and OpenRouter AI.

The system automatically:

* receives loan applications from Google Forms
* analyzes applications using AI
* classifies applicants by risk level
* sends automated email notifications
* stores results in Google Sheets

## Technologies Used

* n8n
* OpenRouter AI API
* Gmail
* Google Sheets
* Google Forms

## Workflow Structure

Google Forms → Google Sheets Trigger → OpenRouter AI → IF Node → Gmail → Google Sheets

## Features

* AI risk analysis
* automated approval/rejection
* workflow automation
* email automation
* data storage automation
